# GoTop

GoTop is a high-performance quantitative trading framework written in Go, providing unified exchange interfaces, data source interfaces, and WebSocket streaming capabilities.

## Features

- Unified Exchange Interface
  - Support for multiple trading types (Spot, Futures, etc.)
  - Standardized order management
  - Unified account management

- Efficient Data Processing
  - WebSocket streaming data handling
  - Support for multiple data sources (CSV, Database, etc.)
  - Real-time market data subscription

- Modular Design
  - Independently extensible exchange modules
  - Flexible data source expansion
  - Fully decoupled strategy modules
